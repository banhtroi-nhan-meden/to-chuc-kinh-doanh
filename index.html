    Code hiện tại của bạn chạy tốt trên desktop nhưng có 5 vấn đề lớn trên mobile: Vấn đề 1: overflow:hidden Hiện tại: body{ font-family:Inter,sans-serif; background:#fff; overflow:hidden; } Trên điện thoại, khi nội dung dài hơn màn hình sẽ bị cắt mất. Sửa: body{ font-family:Inter,sans-serif; background:#fff; overflow-x:hidden; } Vấn đề 2: Font quá lớn Hiện tại: h1{ font-size:58px; } 58px là quá lớn trên màn hình 390px. Đổi sang Responsive Typography: h1{ font-size:clamp(2rem,5vw,4rem); line-height:1.05; } h2{ font-size:clamp(1rem,2.5vw,1.5rem); } ul{ font-size:clamp(.95rem,2vw,1.1rem); } Vấn đề 3: Circle quá to Hiện tại: .circle{ width:320px; height:320px; } 320px gần như chiếm hết chiều ngang điện thoại. Đổi: .circle{ width:min(320px,70vw); height:min(320px,70vw); font-size:clamp(1.2rem,4vw,2.2rem); } Vấn đề 4: Layout 2 cột Hiện tại: .wrap{ grid-template-columns:1.1fr .9fr; } Trên mobile sẽ rất chật. Thêm: @media (max-width:768px){ .wrap{ grid-template-columns:1fr; gap:30px; } .visual{ order:-1; } } Kết quả: Mobile [CIRCLE] TITLE SUBTITLE BULLETS QUOTE đẹp hơn rất nhiều.
